# Project 1 Code Evidence

## 목적

공개 가능한 코드 근거를 연결한다.

## 현재 확정된 사실

- 기존 공개 코드 위치: `portfolio/01-escape-room/code/`.
- 현재 working area에는 코드 원문을 복사하지 않고 공개 코드 위치와 claim mapping을 연결한다.
- 파일별 역할은 `../source-file-map.md`, claim별 근거는 `../claim-evidence-map.md`에서 관리한다.

## 확인 필요

- 필요 시 전체 코드 복사가 아니라 짧은 excerpt 또는 링크 요약만 둔다.
- scene/prefab/Inspector binding, Play Mode 영상, 빌드 파일은 이 폴더에 두지 않는다.

## 과장 위험 / 금지 표현

- 공개 범위가 확인되지 않은 원본 파일을 그대로 추가하지 않는다.
- repo에 없는 지원 타입이나 Unity scene 연결을 코드 근거처럼 쓰지 않는다.
