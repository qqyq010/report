# Public Publication Policy

이 저장소는 공개 포트폴리오 후보 저장소이므로 다음 기준을 적용합니다.

## Public

- 개인 프로젝트의 본인 작성 C# 핵심 파일
- 팀 프로젝트에서 공개 허가를 받은 코드 또는 직접 설명 가능한 짧은 발췌
- 민감 정보가 제거된 README, 구조도, 검증 기준 문서
- 샘플 데이터와 로컬 checker/validator 설명

## Public After Sanitizing

- 팀 프로젝트 작업 로그 요약
- 회의 요약의 일부 문장
- AI/Codex 활용 evidence matrix
- Project3 workflow 문서의 축약본
- 폴더 구조 스크린샷

## Private

- 팀 전체 소스 원본
- PlayFab key, SecretKey, TitleData 실제 운영값, CloudScript 배포 원본
- raw transcript, raw meeting video
- 팀원 이름, 계정, 연락처, 개인 평가 자료
- 유료 에셋 원본
- 회사 또는 팀 내부 의사결정 민감 자료

## Interview Only

- 팀 프로젝트 원본 Unity 소스
- PlayFab 연동 코드 전체
- 작업 로그 원본
- 회의 요약 원본

면접에서는 필요한 파일만 로컬에서 열어 설명하고, 공개 저장소에는 요약과 핵심 구조만 남깁니다.
