## Verification Labels

| Label | Meaning |
| --- | --- |
| Code Evidence | 코드 파일 또는 메서드 구조로 확인 |
| Document Evidence | 문서상 설계 또는 작업 기준으로 확인 |
| Log Evidence | 작업 로그, 검증 로그, 회의 요약으로 확인 |
| Build Evidence | 빌드 또는 self-test 기록으로 확인 |
| Play Mode Evidence | Unity Play Mode에서 직접 확인된 증거 |
| Live Evidence | PlayFab, 서버, 배포 환경에서 확인된 증거 |

코드가 존재한다는 사실을 Play Mode 또는 live 검증 완료로 표현하지 않습니다.
